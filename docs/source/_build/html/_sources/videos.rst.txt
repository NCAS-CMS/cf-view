

cf-view training videos
================

A series of videos showing various features of cf-view

- `Introduction <videos/cfview_intro.mp4>`_ - Make map, vertical cross section, vector and line plots
- `Data transform <videos/cfview_transform.mp4>`_ - Dimension collapses such as zonal or time mean, interpolation to another grid
- `View data <videos/cfview_view_data.mp4>`_ - View and inspect data as a table of values
- `Properties <videos/cfview_properties.mp4>`_- View, modify, delete and add properties to data and dimensions
- `Saving data <videos/cfview_data_writing.mp4>`_ - modify and save data to a netCDF file
